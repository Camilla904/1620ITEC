 # translating English into Pig Latin
 
- Each word is modified by moving the initial sound of the word to the end and adding "ay"
- Words that begin with vowels have the "yay" sound added on the end

> - book ookbay 
> - table abletay 
> - item itemyay 
> - chair airchay
